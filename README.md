# Home Service Content

SEO-optimized content framework for home service businesses (fencing, roofing, landscaping). Templates and guidelines for producing content that ranks, converts, and passes E-E-A-T standards.

## Structure

```
home-service-content/
  CHECKLIST.md          # Pre-publish quality checklist
  STYLE-GUIDE.md        # Voice, tone, writing rules, E-E-A-T signals
  templates/
    service-page.md     # Service page template (content + SEO)
    blog-post.md        # Blog post template (content + SEO)
    location-page.md    # Service area / location page template (content + SEO)
```

## Templates

Each template is split into two sections:

1. **Content** — the actual page copy with structure, CTAs, and trust signals
2. **SEO** — schema markup, keywords, internal/external links, Open Graph tags

| Template | Use For | Word Count Target |
|---|---|---|
| `service-page.md` | Core service offerings (fence installation, roof repair, etc.) | 1,200 - 1,800 |
| `blog-post.md` | Educational and informational content | 1,500 - 2,500 |
| `location-page.md` | City/service area pages with geo-specific content | 800 - 1,200 |

## Usage

1. Pick the right template for your content type
2. Fill in all bracketed `[placeholders]`
3. Run through `CHECKLIST.md` before publishing
4. Follow `STYLE-GUIDE.md` for voice, tone, and writing rules

## SEO Built In

Every template includes pre-structured:

- Schema markup (Service, LocalBusiness, FAQPage, Article, BreadcrumbList)
- Open Graph tags
- Meta title/description format
- URL slug format
- Internal and external linking slots
- NAP consistency block
- areaServed schema for location pages
- AI Overview optimization through direct-answer formatting
