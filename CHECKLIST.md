# Home Service Content Checklist

## Content

- [ ] Format — content in first section, SEO in second
- [ ] E-E-A-T signals included
- [ ] Avoid cliches — no em dashes, no listing 3 things, no AI-sounding phrases, no passive voice, no filler phrases ("in today's world", "look no further")
- [ ] Visuals — tables, charts, or simple graphics to help readers digest content
- [ ] H tags — H1, H2, H3 properly structured
- [ ] Meta title and meta description (formula: [Service] in [City] | [Brand])
- [ ] Word count — service pages and blog posts have different targets
- [ ] Table of contents — for longer blog posts
- [ ] Image alt text

## SEO

- [ ] Schema — FAQ, Service, Page (details pre-filled in code)
- [ ] Local CID link
- [ ] Geo content — for service area / location pages only
- [ ] AI Overview optimized
- [ ] Keywords added within the content
- [ ] Internal links — for both service and blog content
- [ ] External links to authority sites — blog content only
- [ ] URL slug format (e.g., /fence-installation-new-orleans/)
- [ ] Breadcrumb schema
- [ ] Open Graph tags (title, description, image)
- [ ] NAP consistency — Name, Address, Phone same across all pages
- [ ] areaServed in schema — for location/service area pages

## Conversion

- [ ] Primary CTA — Get a Free Quote, Call Now
- [ ] Secondary CTA — Learn More, See Our Work
- [ ] Trust signals — licenses, certifications, years in business, service guarantees
- [ ] Social proof — star ratings, review snippets, project count
- [ ] Before/after section — fence, roofing, and landscaping pages

## Tone and Style

- [ ] Voice — home service expert/contractor; professional but empathetic to homeowner's problem
- [ ] Jargon — homeowner-friendly, not overly technical
- [ ] Readability — sentences under 20 words where possible
- [ ] No overly salesy language — helpful over pushy

## Verification

- [ ] Content information is accurate and up to date
- [ ] Sources cited where applicable (NAHB, industry associations, manufacturer specs)
- [ ] Review/update trigger — flag for refresh if stats are 12+ months old or after a major algorithm update
